# Step Public Services Platform

## 📌 Overview
A full-stack web application designed to provide public services such as university admissions, travel booking, visa applications, and financial transfers.

## 🚀 Features
- User authentication (Login / Signup)
- Admin dashboard with analytics
- Real-time service request system
- Telegram notification integration
- Data visualization using charts
- Export data to Excel

## 🛠 Technologies Used
- Backend: Python (Flask)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Visualization: Chart.js
- 3D Graphics: Three.js
- API Integration: Telegram Bot API

## 📊 Highlights
- Full-stack architecture
- Real-world service platform
- Admin analytics dashboard
- AI-ready structure for future upgrades
